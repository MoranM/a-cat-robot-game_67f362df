# Feline Fury: RoboCat's Revenge

## Game Overview

In a dystopian future, you control RoboCat, a highly advanced feline automaton tasked with eliminating the robotic mouse infestation in CyberCity.

## How to Run the Game

1. Ensure you have Node.js installed on your machine.
2. Clone this repository.
3. Run `npm install` to install all dependencies.
4. Start the game server with `npm start`.
5. Open your browser and go to `http://localhost:8080` to play the game.

## Controls

- Move around using WASD keys.
- Use the mouse to look around and target enemies.

## Goals

- Eliminate all robotic mice.
- Defeat the Mega Mouse Boss to save CyberCity.
